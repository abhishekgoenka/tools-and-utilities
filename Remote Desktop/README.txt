Remote Desktop is the next generation of mRemoteNG, a full-featured, multi-tab remote connections manager.

It allows you to store all your remote connections in a simple yet powerful interface.

Currently, these protocols are supported:

 * RDP (Remote Desktop)
 * VNC (Virtual Network Computing)
 * ICA (Independent Computing Architecture)
 * SSH (Secure Shell)
 * Telnet (TELecommunication NETwork)
 * HTTP/S (Hypertext Transfer Protocol)
 * Rlogin (Rlogin)
 * RAW

Other silent features
 * Connect SSMS if server has SQL Server installed
 * Run SQL Profiler if server has SQL Server installed
